# Web UI 可視化パネル

[← メインドキュメントへ戻る](../../README.ja.md)

---

Nokori にはローカル可視化管理パネルが内蔵されている。一つのコマンドですべての実行状態を確認できる。

```bash
nokori web                    # http://localhost:8765 を自動で開く
nokori web --port 9000        # カスタムポート
nokori web --no-browser       # サーバーのみ起動
```

---

## ページ一覧

| ページ | 内容 |
|------|------|
| **ダッシュボード** | ルール状態別カウント、24h 注入統計、Embed サーバー制御、Gate 状態、抽出待ちジョブ、ライフサイクルエビデンス |
| **ルール** | フィルタ付きリスト、詳細ページ（trigger、action、evidence log、lifecycle evidence、replacement lineage）、編集、退役 |
| **検索シミュレーション** | プロンプトを入力してルールヒットを確認：BM25 + embedding スコア、HOT/WARM 階層、マッチトークン、シャドウプール |
| **アクティビティ — タイムライン** | 全システムイベントストリーム：hook 呼び出し、コールドパイプライン判定、ライフサイクル遷移、事後評価。カラーラベル、結果バッジ、session/タイプフィルタ |
| **アクティビティ — Dashboard** | 運用チャート：イベントソース棒グラフ、コールドパイプライン変換ファネル、エラー円グラフ、エラー傾向折れ線 |
| **注入履歴** | ルール注入のタイムライン。レベル/セッションでフィルタ可能 |
| **抽出パイプライン** | 保留中/完了ジョブ、各 transcript の抽出状態 |
| **ライフサイクル** | candidate → active、active → trusted、suppressed recovery のエビデンス進捗 |
| **設定とヘルス** | 現在の設定値 + 各項目のヘルスチェック |
| **ログ** | WebSocket リアルタイムログストリーム、レベルフィルタ対応 |

## 特徴

- **多言語対応**：ブラウザ言語を自動検出し、日本語/英語/中国語を切り替え可能
- **ダーク/ライトモード**：システムの `prefers-color-scheme` に追従。手動切り替えも可能
- **Embed サーバー制御**：パネル上からローカル embedding サーバーを直接起動/停止
- **アニメーション**：数値カウントアップ、カーソル追従グロー、フローティンググラデーション背景、スタッガー入場

---

## フロントエンド開発

```bash
cd web
npm install
npm run dev          # Vite 開発サーバー :5173、/api を :8765 にプロキシ
# 別のターミナル：
nokori web --no-browser   # API バックエンド起動
```
